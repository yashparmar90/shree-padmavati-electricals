# Shree Padmavati Electricals - Cloned Website

This folder contains a local clone of the "Shree Padmavati Electricals" digital card and website.

## How to Use

1.  **Open the Digital Card:**
    *   Double-click on `index.html` to open the main digital card in your web browser.
    *   All buttons and links should work as expected.
    *   The "Website" button will navigate to the local copy of the website.

2.  **Open the Website Directly:**
    *   Double-click on `website.html` to view the website page directly.

## Files

*   `index.html`: The main digital card page.
*   `website.html`: The linked website page (originally a Next.js app, converted to static HTML).
*   `css/`: Contains stylesheet files.
*   `js/`: Contains JavaScript files (for the digital card).
*   `images/`: Contains all downloaded images and icons.

## Notes

*   The cloned `website.html` is a static version of the original dynamic site. Some complex interactive features (like dynamic form submissions) may not functional exactly as the live site, but the visual layout and content are preserved.
*   The Digital Card `index.html` retains its core functionality including "Save Contact", "Share", and redirection links.
